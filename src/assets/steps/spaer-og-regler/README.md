# Billeder: spaer-og-regler

Læg billeder til dette byggestep i denne mappe.

- Understøttede formater: **.jpg / .jpeg / .png / .webp / .avif**
- Filnavnet bruges som alt-tekst (fx `bundrem-i-vinkel.jpg` → "bundrem i vinkel").
- Billederne vises automatisk i galleriet på step-siden og optimeres ved build.

Ingen kodeændring nødvendig – bare læg filerne her.
